# صور المنتجات — الرفع بدون برمجة

## الطريقة التلقائية (الأسهل) ✅

1. سمِّ ملف الصورة **بمعرّف المنتج بالضبط**: `kahi-balm.png` أو `.jpg` أو `.webp`
   (القائمة الكاملة للأسماء في صفحة `admin-images.html`).
2. ارفعها إلى هذا المجلد عبر GitHub (سحب وإفلات):
   https://github.com/alinasser21001b-blip/adlytic/upload/claude/laughing-hawking-j2u13w/pharmacy/images
3. اضغط **Commit changes** — تظهر الصورة في الموقع تلقائياً خلال دقيقة، بدون تعديل أي كود.

يفضَّل: PNG بخلفية شفافة أو بيضاء، جودة عالية، بدون علامات مائية.

## الطريقة اليدوية (اسم مختلف)

إن أردت اسم ملف مختلفاً، أضف حقل `img` للمنتج في `js/products.js`:

```js
{ id: "kahi-balm", img: "images/اسم-آخر.png", ... }
```

## الشعار

استبدل `logo.svg` بملف شعار الصيدلية الأصلي (بنفس الاسم) ليظهر في الترويسة والبطل والتذييل.
