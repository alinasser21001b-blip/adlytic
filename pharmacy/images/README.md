# صور المنتجات

ضع هنا صور المنتجات الحقيقية (يفضَّل PNG بخلفية شفافة أو بيضاء، جودة عالية).

ثم اربط الصورة بالمنتج في `js/products.js` بإضافة حقل `img`:

```js
{
  id: "kahi-balm",
  img: "images/kahi-balm.png",   // ← أضف هذا السطر
  ...
}
```

ستظهر الصورة تلقائياً في البطاقة وصفحة المنتج والعرض السريع ونافذة الطلب.
بدون صورة، يظهر بديل أنيق بهوية الصيدلية (الرمز التعبيري في حقل `emoji`).
