# فيديو الواجهة السينمائي (Hero)

ضع هنا مقطعاً سينمائياً قصيراً (فيتامينات/عناية) ليظهر خلف عنوان الصفحة الرئيسية.

## كيف تضيفه

1. جهّز مقطعاً قصيراً (يفضّل **6–12 ثانية**، بلا صوت، أفقي 16:9).
2. صدّره بصيغتين إن أمكن وضعهما هنا بالاسمين بالضبط:
   - `videos/hero.mp4`  (H.264 — يعمل على كل المتصفحات)
   - `videos/hero.webm` (اختياري — أخف حجماً)
3. صورة ثابتة تظهر قبل تحميل الفيديو: `images/hero-poster.jpg`.

## نصائح للأداء (مهمة)

- **الحجم الأقصى المستحسن: ~2–4 ميغابايت** حتى لا تبطؤ الصفحة على الجوال.
- دقة 1280×720 كافية (لا حاجة لـ4K في الخلفية).
- المقطع يُشغَّل تلقائياً، صامتاً، ومكرراً (loop) — لذلك لا تضع صوتاً.

## السلوك التلقائي

- إن وُجد الملف: يظهر الفيديو خلف طبقة عاجية شفافة تُبقي النص واضحاً.
- إن لم يوجد: تظهر الخلفية المتدرّجة الأنيقة الحالية تلقائياً — **لا شيء ينكسر**.
- يحترم إعداد «تقليل الحركة» في النظام (لا يشغّل تلقائياً حينها).
