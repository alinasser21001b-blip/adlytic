📸 كيف تضيف صور المنتجات (بدون أي برمجة)

  1. صوّر المنتج أو احصل على صورته الرسمية (يفضَّل خلفية بيضاء أو شفافة PNG).
  2. أعد تسمية الملف بالاسم المطلوب بالضبط كما في القائمة أدناه (مثال: kahi-balm.png).
  3. اضغط الزر الأخضر، ثم اسحب الصور وأفلتها في صفحة GitHub واضغط Commit changes.
  4. خلال دقيقة تظهر الصور تلقائياً في الموقع مكان الرموز — بدون أي خطوة أخرى.
