# صيدلية در الشارقة — الموقع الرسمي

كتالوج رقمي فاخر هدفه تحويل الزوار إلى محادثات واتساب. بدون سلة، بدون دفع إلكتروني،
بدون حسابات — كل مسار في الموقع يقود إلى رسالة واتساب جاهزة ومهنية.

## التشغيل

موقع ثابت بالكامل — افتح `index.html` أو:

```bash
npx serve pharmacy
```

منشور تلقائياً عبر Netlify من فرع `claude/laughing-hawking-j2u13w`
(المجلد `pharmacy`، بدون أمر بناء).

## الإعداد — كل شيء في `js/products.js`

- `STORE_CONFIG.whatsappNumber` — رقم الواتساب بالصيغة الدولية (حالياً `9647710595805`
  من منشورات الصيدلية — حدّثه إن كان رقم الواتساب مختلفاً).
- `STORE_CONFIG.phones` — الأرقام الظاهرة في التذييل.
- `CATEGORIES` — التصنيفات.
- `PRODUCTS` — أضف منتجاً واحداً وسيظهر تلقائياً في الكتالوج وصفحته وأقسام الترشيح.

### إضافة صور حقيقية

ضع الصورة في `images/` وأضف `img: "images/الاسم.png"` للمنتج — التفاصيل في `images/README.md`.

## رحلة الطلب

1. العميل يكتشف المنتج (بحث، تصنيفات، ترشيحات).
2. يقرأ صفحة المنتج: وصف، فوائد، مكونات، استخدام، لمن يناسب، تنبيهات.
3. يضغط «اطلب عبر واتساب» ← تفتح **نافذة طلب راقية** تجمع: الاسم، الهاتف (اختياري)،
   المدينة، الملاحظات — وتتذكر بياناته لطلباته القادمة.
4. يفتح واتساب برسالة عربية جاهزة تتضمن اسم المنتج وبيانات العميل ورابط الصفحة.

## قرارات تصميمية مقصودة

| القرار | السبب |
|---|---|
| **لا أسعار معروضة** | الصيدلية لا تنشر أسعارها؛ «اسأل عن السعر» يولّد محادثة بدل أن يخسرها تغيّر الأسعار. |
| **لا تقييمات وهمية** | الثقة أساس عمل الصيدلية — أبرزنا Highlights حقيقية بدلاً من مراجعات مخترعة. |
| **نافذة طلب قبل واتساب** | رسالة تحمل الاسم والمدينة تختصر على الصيدلي نصف المحادثة وتُشعر العميل بالجدية. |
| **حفظ بيانات العميل محلياً** | الطلب الثاني يصبح بضغطتين — تكرار الشراء أسهل. |
| **وصف حذر بلا ادعاءات طبية** | ما لم نتأكد منه تُرك خارجاً؛ عبارات «استشر الصيدلي» تبني مصداقية. |
| **باقات منسّقة** (طلاب، حمل) | تحاكي منشورات الصيدلية الفعلية وترفع قيمة الطلب الواحد. |
| **بديل صور بهوية الصيدلية** | الموقع يعمل كاملاً اليوم، والصور الحقيقية تُضاف تدريجياً دون كسر أي شيء. |

## المزايا التحويلية

شريط ثقة، عدّادات متحركة، بحث فوري مع هياكل تحميل، عرض سريع (Quick View)،
مفضلة ♥ وسجل «شاهدتها مؤخراً» (localStorage)، بانر استشارة مجانية، شريط طلب ثابت
على الجوال، فقاعة واتساب عائمة، مشاركة عبر Web Share، بحث فارغ يتحول لمحادثة،
JSON-LD لنتائج غوغل، ودعم كامل للعربية RTL بخطي Tajawal وAmiri.

